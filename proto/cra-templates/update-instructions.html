(Bi-)Weekly page updates

On weeks with a CRB update there are 18 pages that need to be updated. CRB has bi-weekly updates whereas CRSB and CRCB need to be updated weekly.

  • For a CRB week = 3 pages x 3 benefits x 2 languages = 18 pages

Pages that need to be updated (bi-)weekly:

  • https://www.canada.ca/en/revenue-agency/services/benefits/recovery-benefit/crb-periods-apply.html
  • https://www.canada.ca/en/revenue-agency/services/benefits/recovery-benefit/crb-how-apply.html
  • https://www.canada.ca/en/revenue-agency/services/benefits/recovery-benefit/crb-return-payment.html
